Introduction to Arrays

What is an array ?

An array is a data structure that stores A collection of elements typically obtained data type array are used to store multiple values in the single variable which allows for efficient data management and manipulation

Characteristics of Arrays

  • Fixed Size:The size of an array is defined when it is created and cannot be changed.
  • Indexed:Each element in an array is accessed using an index,which typically starts from 0.
  • Homogeneouse Elements:all elements in an array must be of the same data type such as int or char.

Types of arrays

  1. One-dimentional arrays: A linear list of elements where each element can be accessed by single index
    • example: int numbers[5] : stores 5 integers
  2. Mighty dimensional arrays : arrays with more than one dimension commonly used for matrices and tables
    • int matrix[3][3] : a 3*3 matrix of integres
  3. Dynamic arrays : arrays that can change size during program execution typically implemented using data structures like list

Characteristics of tree

  • Child node: Unload that is a descendant of another node
  • Parent node: A node that has one or more child node
  • Leaf node: A node that has no children

Introduction To Graphs

A graph is the data structure that consists of sets of vertices and a set of edges that connects pair of vertices graphs are used to represent relationship or connection between various entities and are fundamental in computer science for modeling complex networks such as social networks transportation systems and communication network

Characteristics of graph

  • Vertices or notes the fundamental unit of graph representing entities in graph
  • Edges connection between pair of vertices which can be directed or undirected
  • Degree the number of edges connected to a vertex for directed graph a vertex can have an in degree and one out degree
  • Part a sequence of edges that connects a sequence of vertices
  • cycle a path that starts from and end at same vertex without repeating any edges

Types Of Graphs

  1. directed graph
  2. Undirected graph
  3. cyclic graph
  4. acyclic graph
  5. weighted graph
  6. unweighted graph

Applications Of Graph

  1. Social networks: graph are used to model relationships and interaction between individuals or entities in social media platform
  2. transportation networks: represent roads interactions in routes in transportation system helping optimize travel and logistics
  3. web page link analysis: the structure of web can be represented as graph which web pages as vertices and hyperlinks edges enabling search engine to analyze and rank the website
  4. network flow: graphs are utilized in flow networks to model the flow of resources such as electricity or data through network
